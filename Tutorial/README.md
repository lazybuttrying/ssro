# Tutorial

This folder provides a guided walkthrough for SSRO.

The main repository README explains the architecture and project motivation.
The files in this folder explain how to actually step through the starter pipeline.

## Tutorial contents

- `README.md` — overview of the tutorial path
- `01_quickstart.md` — install, run, test
- `02_outputs_walkthrough.md` — how to read the generated artifacts
- `03_how_to_extend.md` — how to modify the starter for your own topic

## Who this is for

This tutorial is for readers who want to understand SSRO as:
- a research systems prototype
- a multi-agent workflow for social science research
- a starter repo they can adapt to labor, finance, policy, or inequality topics

## Recommended order

1. Read the repository root `README.md`
2. Follow `01_quickstart.md`
3. Inspect `02_outputs_walkthrough.md`
4. Use `03_how_to_extend.md` to adapt the project
