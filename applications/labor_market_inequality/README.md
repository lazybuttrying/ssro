# Labor Market Inequality application package

This package is the first topic-specific application of SSRO.

## Research focus

How might generative AI reshape skill demand and inequality in labor markets?

## Current status

The current implementation is a starter example based on synthetic job-postings data.
It demonstrates the workflow rather than making a substantive empirical claim.

## Core components used

- governance layer for the research brief
- literature and theory mapping
- data construction
- measurement registry
- design registry
- descriptive analysis
- reproducibility, robustness, and validity audit

## Main outputs

- `outputs/measurement_registry.json`
- `outputs/design_registry.json`
- `outputs/research_entities.json`
- `outputs/working_note.md`
- `outputs/figure_wage_by_ai_related.png`

## Better next step

Replace the synthetic data with real job-postings data and implement a stronger design such as event study or difference-in-differences.
