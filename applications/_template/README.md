# Application package template

Use this folder as a template for a new SSRO application package.

Include at least:
- research question
- data source description
- measurement notes
- design notes
- expected outputs
- validity concerns

Suggested package layout:
- applications/your_topic/README.md
- applications/your_topic/measurement_notes.md
- applications/your_topic/design_notes.md

Rule of thumb:
Keep the core Research OS stable.
Change the application package by changing the research question, raw data, measurement rules, and design memo.
